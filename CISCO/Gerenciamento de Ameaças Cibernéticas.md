
## GOVERNANÇA E CONFORMIDADE

### Títulos de governança da Segurança Cibernética

| Titulo                       | Função                                                                                                                                                                         |                            |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------- |
| Proprietário dos dados       | Garante a conformidade com politica e procedimentos, atribui a classificação adequada aos ativos de informações e determina os critérios para acessar os ativos de informações |                            |
| Controlador de dados         | Determina as finalidades para as quais e maneira em que os dados pessoais são processados                                                                                      |                            |
| Processador de dados         | Processa dados pessoais em nome do controlador de dados                                                                                                                        | Pessoa ou Empresa          |
| Custódia de dados            | Implementa a classificação e os controles de segurança dos dados de acordo com as regras definidas pelo proprietário dos dados                                                 | Controle técnico dos dados |
| Administrador de dados       | Garante que os dados atendam as necessidades comerciais de uma empresa e atenda aos requisitos regulamentares                                                                  |                            |
| Oficial de proteção de dados | Supervisiona a estratégia de proteção de dados de uma empresa                                                                                                                  |                            |
### Politica de Segurança Cibernética

| Tipos                                      | Descrição                                                                                                                                                                                                                                                        |
| ------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Politica de identificação e autentificação | Especifica quem deve ter acesso aos recursos de rede e quais procedimentos de verificação estão em vigor para facilitar isso                                                                                                                                     |
| Politica de senhas                         | Define os requisitos mínimos de senha, como número e tipo de caracteres usados, e a frequência com que eles precisam ser alterados                                                                                                                               |
| Politica de Utilização Aceitavel           | Destaca um conjunto de regras que determinam o acesso e o uso dos recursos de rede. Ela também pode definir as consequências das violações de política                                                                                                           |
| Politica de acesso remoto                  | Define como se conectar remotamente à rede interna de uma empresa e explica quais informações são acessíveis remotamente                                                                                                                                         |
| Politicas de Manutenção de Rede            | Descreve os procedimentos para atualizar os sistemas operacionais específicos de uma empresa e os aplicativos de usuário final                                                                                                                                   |
| Politica de tratamento de incidentes       | Fornece orientação sobre como relatar e responder a incidentes relacionados à segurança em uma empresa                                                                                                                                                           |
| Politica de dados                          | Define regras mensuráveis para o processamento de dados dentro de uma empresa, como especificar onde os dados são armazenados, como os dados são classificados (alto, médio, baixo, confidencial, público ou privado) e como os dados são tratados e descartados |
| Politica de credenciais                    | Aplica as regras de composição de credenciais, como o comprimento mínimo e máximo de uma senha                                                                                                                                                                   |
| Politica Corporativo                       | Fornece orientações sobre como o trabalho deve ser realizado em uma organização. Os exemplos podem incluir políticas de gerenciamento de mudanças, controle de mudanças ou gerenciamento de ativos                                                               |

### Estrutura de gerenciamento de Segurança de T.I

#### Domínios de Segurança Digital (ISO/IEC 27000)

| 12 Domínios                                                       | Descrição                                                                                                                                                   |                                                   |
| ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| Avaliação de Risco                                                | Determina o valor quantitativo e qualitativo de risco relacionado a uma situação especifica ou ameaça reconhecida                                           |                                                   |
| Politica de Segurança                                             | Documento que aborda as limitações e os comportamentos dos membros de uma empresa                                                                           | Especifica qual dado pode ser acessado e por quem |
| Organização da Segurança da Informação                            | Modelo de governança estabelecido por uma empresa                                                                                                           |                                                   |
| Gerenciamento de ativos                                           | É um inventário do esquema de classificação dos ativos da informação                                                                                        |                                                   |
| Segurança de Recursos Humanos                                     | Contempla os procedimentos de segurança no que diz respeito aos funcionários que são admitidos, transferidos e demitidos de uma empresa.                    |                                                   |
| Segurança física e ambiental                                      | Refere-se à proteção física das informações e das instalações de uma empresa.                                                                               |                                                   |
| Gerenciamento de operações e comunicações                         | Descreve a gestão de controles técnicos de segurança em sistemas e redes                                                                                    |                                                   |
| Aquisição, desenvolvimento e manutenção de sistemas da informação | Refere-se a segurança como parte integrante dos sistemas de informação de uma empresa                                                                       |                                                   |
| Controle de Acesso                                                | Descreve como uma empresa restringe os direitos de acesso a redes, sistemas, funções de aplicativos e dados para impedir acesso de usuários não autorizados |                                                   |
| Gerenciamento de incidentes de Segurança da Informação            | Descreve como antecipar e responder a violações de Segurança da Informação                                                                                  |                                                   |
| Gerenciamento de continuidade de negócios                         | Descreve a capacidade de uma empresa de proteger, manter e recuperar atividades essenciais aos negócios após uma interrupção nos sistemas de informação     |                                                   |
| Comunidade                                                        | Descreve o processo de assegurar a conformidade com as politicas, normas e regulamentos de Segurança da Informação                                          |                                                   |


> [!important] ISO 
>  Usa domínios e não camadas para descrever as categorias
>  É um modelo não hierarquico (Cada modelo tem uma relação direta)
> 

#### Framework da Força de Trabalho de Cibersegurança Nacional
 
 -  O National Institute of Standards and Technologies (NIST) criou a National Cybersecurity Workforce Framework para apoiar organizações que buscam profissionais de segurança digital
 
 A estrutura organiza o trabalho de segurança digital em 7 categorias:

| Categorias               | Descrição                                                                                                                                                         |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Operar e Manter          | Proporciona o suporte, a administração e a manutenção necessários para garantir a segurança e o desempenho do sistema de TI                                       |
| Proteger e Defender      | Identifica, analisa e mitiga ameaças a redes e sistemas internos                                                                                                  |
| Investigar               | Investiga eventos de segurança digital e ataques digitais que envolvem recursos de TI                                                                             |
| Coletar e Operar         | Faz atividades de operações de negação e fraude especializadas e a coleta de informações de segurança cibernética                                                 |
| Analise                  | Realiza revisão e avaliação altamente especializada das informações de segurança cibernética recebidas para determinar sua utilidade para a inteligência          |
| Supervisionar e Governar | Fornece liderança, gerenciamento, direção ou desenvolvimento e advocacia para que uma organização possa conduzir efetivamente o trabalho de segurança cibernética |
| Provisão Segura          | Conceitua, projeta, adquire ou cria sistemas de TI seguros                                                                                                        |
